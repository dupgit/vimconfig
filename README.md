Simple script to generate my vim configuration.

* Installs French spell checking
* Installs some vim plugins :
 * pathogen
 * vim-commentary
 * vim-fugitive
 * vim-sensible
 * vim-endwise
 * vim-surround
 * vim-repeat
 * vim-airline and vim-airline-themes
 * mru
 * nerdtree
 * syntastic
 * vim-polyglot
* Installs onedark color theme
* Installs my .vimrc file
